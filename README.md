# cc-agent-teams

Claude Code のマルチエージェントチーム機能を活用して作成した、日本のサイバーセキュリティ・サイバー安全保障に関する調査レポートです。

## レポート概要

**[japan-cybersecurity-report.md](./japan-cybersecurity-report.md)** - 日本のサイバーセキュリティ・サイバー安全保障 最新動向レポート

- **調査対象期間**: 2024年〜2026年2月
- **全11セクション / 約1,400行**

### セクション構成

| No. | セクション |
|-----|----------|
| 1 | エグゼクティブサマリー |
| 2 | サイバーセキュリティ政策・法制度の動向 |
| 3 | サイバー脅威の現状 |
| 4 | 敵対国の攻撃アクター・攻撃手法と日本へのリスク |
| 5 | サイバー安全保障・防衛体制 |
| 6 | 欧米・友好国のサイバーセキュリティ動向と日本への影響 |
| 7 | 金融機関・金融ITサービス事業者への示唆 |
| 8 | 重要インフラ事業者・ステークホルダーへの示唆 |
| 9 | 新興技術がサイバーセキュリティに与える影響（AI・量子） |
| 10 | 課題と展望 |
| 11 | 参考情報源 |

## 作成方法

本レポートは [Claude Code](https://docs.anthropic.com/en/docs/claude-code) の **マルチエージェントチーム機能（TeamCreate）** を使用して作成しました。

### エージェント構成

```
team-lead（リーダー）
├── ai-security-researcher    … AI・サイバーセキュリティの調査
└── quantum-security-researcher … 量子技術・サイバーセキュリティの調査
```

- 2つの調査エージェントが **並行して** Web検索・調査・執筆を実行
- リーダーエージェントが調査結果を統合し、既存レポートへの挿入・リナンバリング・整合性チェックを実施
